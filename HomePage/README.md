# Home Page Structure

## Sections (in order)
1. **Announcement Bar** - Marquee with promotions
2. **Header** - Logo, nav, icons
3. **Hero Carousel** - Auto-rotating banner (3-5 slides)
4. **Brand Category Grid** - 3 collections (Jewellery Sets, Necklace, Earrings)
5. **Shop by Category** - 3 category thumbnails
6. **Brand Carousel** - Jewellery-Sets promotion banner
7. **About Section** - Brand philosophy text
8. **Video Section** - HTML5 video placeholder
9. **Footer** - Contact, policies, social links

## Image Upload/Delete Functionality
- Admin can upload/manage hero images
- Admin can upload/manage category images
- Delete functionality with confirmation
- Images stored in: /public/images/heroes/, /public/images/collections/

## Image Slots
| Section | Location | Count |
|---------|----------|-------|
| Hero Slides | Carousel | 3-5 |
| Category Images | Grid (3 cols) | 3 |
| Featured Banner | Full width | 1 |
